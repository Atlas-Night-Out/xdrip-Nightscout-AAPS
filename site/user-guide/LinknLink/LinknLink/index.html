    Intro

    Full Protocol Support and Ease of Use iSG is a favourably compatible smart home gateway supporting all Global IoT Protocols, including Wi-Fi, Matter, zigbee,z-wave, IR, RF, BLE

    Zigbee and z-wave are supported via USB Dongle, IR and RF are supported by eHub), enabling almost all smart home devices connected and managed through iSG regardless of brand or protocol.

    Global IoT Protocols


    Items you are going to need

    1. LinknLink iSG (The Next Generation All-in-One Super Smart Home Gateway)
    2. Home Hub Mount Will Only need if you are going to fix to a wall!
    3. Atolla 4 ports USB 3.0 Hub SuperSpeed Data Transmission with On Off Switch + 1 Charging Port with 15W(5V/3A) Powered Supply Adapter and 1M USB 3 Extension Cable If you are needing to extend the USB lead, to power it on.
    4. UGREEN USB to USB C Cable Right Angle 3A 90 Degree USB C Charger Cable Fast Charging Compatible with iPhone 15 Pro Max Galaxy S24 S23 Ultra A53 Pixel 7 6 Redmi Note 11 Huawei P60 P50 Pro (3M)
