Xdrip Follower Setup

If you would like to follow these instructions with video then click image below other wise continue step by step below

First, we need to setup the Master mode

in Sync Settings on your main phone

1. Go to settings



2. then xdrip+ sync settings



3. These are the Settings to select.



Now display the security key QR code and scan with our Followers handset


  1. Go to show settings QR codes on main mobile



  2. And select xdrip plus security key settings only.



Now configure the follower’s handset using its camera

  1. Go to settings / hardware data source



  2. and then select xdrip+ sync follower



  3. Now select Configure and allow permissions.



  4. Then select Auto Configure again



  5. and scan the other mobile bar code, with it



If you need to setup mulitiple xdrip Followers then see my other setup on how to do this. I will add a link

when I have done it!
