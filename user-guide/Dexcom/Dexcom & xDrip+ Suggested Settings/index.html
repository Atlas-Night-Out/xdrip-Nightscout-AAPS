    Dexcom & xDrip+ Suggested Settings


    Dexcom One, Dexcom G6, G7, and One+ collector suggested settings For ideal performance with xDrip+ and Dexcom devices, follow these advised settings.

    If you're completely new to xDrip+ and you want to install it for the first time continue to Download xdrip Or Install xdrip

    Bluetooth Settings⌁

    • Confirm Settings −> Less common settings -> Bluetooth Settings

    G5/G6/G7/ Dex1 Debug Settings⌁

    • Confirm Settings −> G5/G6/G7/ Dex1 Debug Settings -> G5/G6/G7/ Dex1 Debug Settings

    Other misc options⌁

    • Confirm Settings −> Less common settings -> Other misc options

    Aggressive service restarts⌁

    • Enable Settings −> Less common settings -> Aggressive service restarts

    Automatic Calibration⌁

    • Disable Settings −> Less common settings -> Advanced Calibration -> Automatic Calibration
